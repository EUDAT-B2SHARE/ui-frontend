# B2SHARE frontend

B2share 2.0 UI, powered by [AngularJS 1.3](https://angularjs.org), and various other thirdparty opensource projects.

### Prerequisites

* Nodejs
* npm


### Installation

```bash
git clone git@github.com:EUDAT-B2SHARE/ui-frontend.git
cd ui-frontend/
npm install
```

### Running

```bash
npm start
```
