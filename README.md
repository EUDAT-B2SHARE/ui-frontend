# B2SHARE frontend

B2share 2.0 UI, powered by [AngularJS 1.3](https://angularjs.org), and various other thirdparty opensource projects.

### Prerequisites

* Nodejs
* npm


### Installation

```bash
git clone https://git.osd.surfsara.nl/EUDAT-B2SHARE/b2share-frontend.git
cd b2share-frontend/
npm install
```

### Running

```bash
npm start
```
